# Tic-Tac-Toe
🎮 Tic Tac Toe Game  A modern and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. It features a dynamic 3x3 game board, player turn tracking, win/draw detection, restart functionality, and a light/dark theme toggle. The UI includes smooth animations, gradient backgrounds, and a responsive design for an enhanced user experience.
