# 🎮 Tic Tac Toe Game

A modern, responsive, and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**. This project demonstrates core front-end development concepts such as DOM manipulation, event handling, game logic implementation, and UI/UX design.

---

## 🚀 Features

- 🎯 **Interactive 3x3 Game Board**
  - Dynamically generated grid using JavaScript
  - Smooth click interactions for each cell

- 👥 **Two-Player Mode**
  - Alternating turns between Player X and Player O
  - Real-time turn indication

- 🏆 **Win Detection System**
  - Checks all possible winning combinations
  - Displays winner instantly

- 🤝 **Draw Detection**
  - Detects when all cells are filled without a winner

- 🔄 **Restart Game**
  - Reset button to start a new game anytime

- 🌗 **Light/Dark Theme Toggle**
  - Switch between themes with a single click
  - Uses CSS variables for dynamic styling

- 🎨 **Modern UI Design**
  - Glassmorphism effect with blur
  - Animated gradient background
  - Hover animations for better user experience

- 📱 **Responsive Design**
  - Works smoothly on desktop and mobile devices

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the application  
- **CSS3** – Styling, animations, and responsive design  
- **JavaScript (Vanilla)** – Game logic and interactivity

## ⚙️ How It Works

1. The game board is dynamically created using JavaScript.
2. Players take turns clicking on empty cells.
3. After each move:
   - The game checks for a winning combination.
   - If no winner, it checks for a draw.
4. The game status is updated in real time.
5. Users can restart the game or switch themes anytime.

---

## 🎮 Game Rules

- The game is played on a 3×3 grid.
- Player X goes first, followed by Player O.
- The first player to align 3 marks (row, column, or diagonal) wins.
- If all cells are filled without a winner, the game ends in a draw.
